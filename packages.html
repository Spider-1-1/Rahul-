7. Write a Java program to perform employee payroll processing using packages. In the java file, Emp.java creates a package employee and creates a class Emp. Declare the variables name,empid, category, bpay, hra, da, npay, pf, grosspay, incometax, and allowance. Calculate the values in methods. Create another java file Emppay.java. Create an object e to call the methods to perform and print values.

Code:
                    
                        package employee;

                        public class Emp {
                            private String name;
                            private int empId;
                            private String category;
                            private double basicPay;
                            private double hra;
                            private double da;
                            private double netPay;
                            private double pf;
                            private double grossPay;
                            private double incomeTax;
                            private double allowance;
                        
                            public Emp(String name, int empId, String category, double basicPay) {
                                this.name = name;
                                this.empId = empId;
                                this.category = category;
                                this.basicPay = basicPay;
                                calculatePay();
                            }
                        
                            private void calculatePay() {
                                hra = basicPay * 0.20; // 20% of basic pay
                                da = basicPay * 0.10; // 10% of basic pay
                                allowance = basicPay * 0.05; // 5% of basic pay
                                grossPay = basicPay + hra + da + allowance;
                                pf = grossPay * 0.12; // 12% of gross pay
                                incomeTax = grossPay * 0.10; // 10% of gross pay
                                netPay = grossPay - (pf + incomeTax);
                            }
                        
                            public void displayPayroll() {
                                System.out.println("Employee Name: " + name);
                                System.out.println("Employee ID: " + empId);
                                System.out.println("Category: " + category);
                                System.out.println("Basic Pay: " + basicPay);
                                System.out.println("HRA: " + hra);
                                System.out.println("DA: " + da);
                                System.out.println("Allowances: " + allowance);
                                System.out.println("Gross Pay: " + grossPay);
                                System.out.println("PF: " + pf);
                                System.out.println("Income Tax: " + incomeTax);
                                System.out.println("Net Pay: " + netPay);
                            }
                        }                                        
                    
                    
Output:

                    
                        Employee Name: John Doe
                        Employee ID: 101
                        Category: Developer
                        Basic Pay: 50000.0
                        HRA: 10000.0
                        DA: 5000.0
                        Allowances: 2500.0
                        Gross Pay: 62500.0
                        PF: 7500.0
                        Income Tax: 6250.0
                        Net Pay: 47500.0                                           
                    
                    

8.Create a Java program to create a package with public class and protected members to be accessed in another class.

Code:
                    
                        package myPackage;

                        public class Employee {
                            protected String name;
                            protected int empId;
                        
                            public Employee(String name, int empId) {
                                this.name = name;
                                this.empId = empId;
                            }
                        
                            protected void displayInfo() {
                                System.out.println("Employee Name: " + name);
                                System.out.println("Employee ID: " + empId);
                            }
                        }
                        import myPackage.Employee;

                        public class Main extends Employee {
                            public Main(String name, int empId) {
                                super(name, empId);
                            }
                        
                            public static void main(String[] args) {
                                Main emp = new Main("Alice", 102);
                                emp.displayInfo();
                            }
                        }                                          
                    
                    
Output:

                    
                        Employee Name: Alice
                        Employee ID: 102
