1. Create an applet showing the string "Hello World" which is embedded in the HTML page.

Code:

7. Create an applet in Java to draw 4 concentric circles.


Code:
            import java.applet.Applet;
            import java.awt.Graphics;
            
            public class ConcentricCirclesApplet extends Applet {
                public void paint(Graphics g) {
                    for (int i = 0; i < 4; i++) {
                        g.drawOval(50 - (i * 20), 50 - (i * 20), 100 + (i * 40), 100 + (i * 40));
                    }
                }
            }
                    
Output:
The applet displays 4 concentric circles.

13. Create an applet in Java to set font.


Code:

                public void paint(Graphics g) {
                    Font font = new Font("Arial", Font.BOLD, 24);

                    g.drawString("Custom Font!", 20, 30);
                }
            }
                    
Output:
The applet displays text with a custom font.


14. Create an applet to display a digital clock.


Code:
            import java.applet.Applet;
            import java.awt.Graphics;
            import java.util.Calendar;
            
            public class DigitalClockApplet extends Applet {
                public void paint(Graphics g) {
                    Calendar now = Calendar.getInstance();
                    String time = String.format("%02d:%02d:%02d", now.get(Calendar.HOUR_OF_DAY), now.get(Calendar.MINUTE), now.get(Calendar.SECOND));
                    g.drawString(time, 20, 20);
                }
            }
                    
Output:
The applet displays the current time in digital format.


15. Java Program to Draw a Smiling Face using Graphics Class Methods.


Code:
            import java.applet.Applet;
            import java.awt.Graphics;
            
            public class SmilingFaceApplet extends Applet {
                public void paint(Graphics g) {
                    g.drawOval(50, 50, 150, 150); // Face
                    g.drawOval(90, 90, 15, 15);   // Left Eye
                    g.drawOval(140, 90, 15, 15);  // Right Eye
                    g.drawArc(90, 100, 70, 70, 0, -180); // Mouth
                }
            }
                    
Output:
The applet displays a smiling face.
